# Pseudo Markets PostgreSQL Infrastructure

This folder is reserved for shared PostgreSQL infrastructure assets used by the Pseudo Markets NextGen platform.

The first service planned to rely on this shared PostgreSQL infrastructure is `pseudomarkets-nextgen-transaction-processing`.

Examples of files that can live here in the future:

- initialization scripts
- seed scripts
- shared migration helpers
- backup or restore helpers

The initial transaction processing scaffold does not require shared PostgreSQL scripts yet, so this folder is currently just a reserved location for future platform-level Postgres assets.
